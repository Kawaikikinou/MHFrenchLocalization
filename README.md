Marvel Heroes Omega Localization into French

To change the .string files use MHString2JSON => https://github.com/mikwaldo/MHString2JSON/releases/tag/Release
It allows to convert .string to json and json to .string

To change achievements translation use MHAchievementsTranslator => https://github.com/Kawaikikinou/MHAchievementsTranslator
Add a language fr_fr to translate in french
